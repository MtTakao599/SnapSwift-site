## Features

### 📅 Timeline & Folders
Browse photos organized by date or folder. Instant slideshow from any folder.

### 🔍 Similar Image Search
Find near-duplicate photos using dHash/pHash. Processes 3,000 images in ~0.1 seconds.

### 🗺️ GPS Mapping
Import Google Timeline or GPX files and pin your photos on a map. KML export supported.

### 🪪 ID Photo Creator
Create passport, resume, and license photos with AI background removal, face tilt correction, skin smoothing, and L-size print layout — all on-device.

### 🔄 Batch Format Conversion
Convert to WebP, JPG, PNG, GIF, or APNG in bulk. HEIC and AVIF input supported.

### 🎨 15 Filters
Instagram-style filters including Clarendon, Lark, Juno, Valencia, plus B&W, Sepia, and High Contrast.

### 🔒 EXIF Privacy
Fully delete or edit EXIF data (date, author, GPS, copyright) to protect your privacy.

### 🖼️ Format Support
| Format | Thumbnail | Viewer | Conversion |
|--------|-----------|--------|------------|
| JPEG / PNG / WebP / BMP | ✅ | ✅ | ✅ |
| HEIC / HEIF | ✅ | ✅ | ✅ |
| AVIF | ✅ | ✅ | ✅ |
| GIF | ✅ | ✅ | ✅ |
| SVG | ✅ (Windows) | ✅ | — |
| PDF | ✅ | ✅ (scroll) | — |
| MP4 / MOV / WebM / M4V | ✅ | ✅ | — |

### ⚡ Performance
- Thumbnail cache with BLAKE3 hashing
- 5,000+ photos browsable without lag
- Dynamic import for heavy libraries (TensorFlow, MediaPipe)

## Privacy

- **No data collection** — all processing happens on your device
- **No ads** — completely free, no monetization
- **No cloud** — your photos never leave your device
- Exception: "Search with Google Lens" sends the selected image to Google
