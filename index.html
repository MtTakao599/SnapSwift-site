Linux版について

v1.3.1ではLinux版を安定化し、ディストリビューション差を整理しました。

配布と推奨環境

Linux版は Ubuntu 24.04 でビルドした AppImage を基準に、Ubuntu / Linux Mint / CachyOS で動作確認しています。Fedoraでは AppImage より RPM版を推奨します。

HEIC/HEIFや一部動画形式では、OS側の ffmpeg / libheif / GStreamer 環境に影響される場合があります。Fedoraでは RPM Fusion / freeworld ffmpeg が必要になる場合があります。

Ubuntu / Debian
sudo apt install ffmpeg xdg-utils \
  gstreamer1.0-plugins-base \
  gstreamer1.0-plugins-good \
  gstreamer1.0-plugins-bad \
  gstreamer1.0-plugins-ugly \
  gstreamer1.0-libav
Arch / CachyOS
sudo pacman -S ffmpeg xdg-utils \
  gstreamer gst-plugins-base \
  gst-plugins-good gst-plugins-bad \
  gst-plugins-ugly gst-libav
Fedora
# RPM版推奨
# HEIC/動画で必要に応じて
# RPM Fusion / freeworld ffmpeg を導入

PDF表示について

PDF表示には PDFium を使用しています。v1.3.1ではサイズ上限を拡張し、最大5GBまでのPDFを警告付きで試行できます。120MBを超えるPDFでは、表示に時間がかかる、またはメモリ不足や互換性の問題で開けない場合があります。

一部の特殊なPDFでは、Adobe Acrobat Readerで開けてもSnapSwiftでは開けない場合があります。

プライバシー

安心して写真を扱えるよう、広告や解析を入れず、端末内処理を重視しています。

個人情報を収集しません

SnapSwift はお客様の個人情報を収集しません。写真データは基本的にお客様のデバイス内で処理されます。

広告・解析なし

広告表示や利用状況解析ツールは使用していません。無料のまま、シンプルに使えます。

外部サービス

Googleレンズなど外部サービスを使う機能では、利用時のみ外部サービスへ遷移します。
